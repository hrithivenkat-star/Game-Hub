🎮 How 2 Player Mode Works
01

Pick a mode

2 Players: share keyboard/screen. vs AI: play solo, choose difficulty.

02

Keyboard split

P1 always uses WASD / W key. P2 uses Arrow keys. Turn-based games share the same screen.

03

Mobile split

Left half of screen = P1 touch. Right half = P2 touch. On-screen buttons shown for each.

04

Tournament

7 random games, 1 point per win. Running score shown after each game. Trophy at the end!
