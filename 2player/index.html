🎮 How 2 Player Mode Works
01

Pick a mode

Select 2 Players for local same-device battles, or vs AI to play solo against the machine.

02

Split keyboard

P1 uses WASD or W. P2 uses Arrow keys. Turn-based games let both players tap the same screen.

03

Mobile controls

Every game shows on-screen touch buttons. Left side = P1, right side = P2 for action games.

04

Tournament

7 random games, 1 point per win. Scoreboard updates live. Trophy ceremony at the end!
